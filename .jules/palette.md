## $(date +%Y-%m-%d) - [Add Clear Search Button Micro-UX]
**Learning:** Adding a small `XIcon` to clear input values greatly improves UX for recurring searches in search bars. However, when placing an absolute positioned element inside an input container, it is critical to add sufficient padding (e.g. `pr-8`) to the input element so the text doesn't overlap the icon. Furthermore, rendering logic must ensure the clear button doesn't conflict with or overlap other UI states, like loading spinners.
**Action:** When implementing clear buttons inside inputs, always verify that text padding (`pr-X`) accounts for the icon width, conditionally hide the button during loading states to avoid layout collisions, and ensure keyboard accessibility through `aria-label` and focus styling.
